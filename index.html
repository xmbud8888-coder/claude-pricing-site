Prompt Caching

      常见问题

      中国大陆可以直接订阅 Claude 吗？

      Anthropic 目前不向中国大陆提供服务，注册需要受支持地区的手机号，支付需要海外发行的银行卡。这是官方政策限制，本站不提供绕过方法。如果你在受支持地区有合法的居住、工作或收款关系，按该地区正常订阅即可。

      为什么网页版比 App Store 便宜？

      苹果对应用内购抽成 15–30%，Anthropic 与多数订阅制产品一样把这部分成本反映在 iOS 内购价上。在 claude.ai 网页端订阅后，App 内直接登录使用，功能没有区别。iOS 各档具体挂牌价以管线核验后的渠道价差板块为准。

      换区买便宜订阅到底有什么风险？

      三层风险：① 商店条款——虚构账单地址违反 Apple 服务条款，账号可能被限制；② 支付——换区后原支付方式失效，需持续购买该区礼品卡；③ 服务——Anthropic 按账号所在地区提供服务，异常支付特征可能触发风控。App Store 价含苹果抽成，相对网页年付 $17/月，省下的差价通常不值得这些风险。

      订阅和 API 怎么选？

      固定月费换固定额度 = 订阅；按量付费 = API。经验法则：日常对话 + 编码助手用订阅（Pro/Max），可预估、可控；生产系统、批量任务、自建产品用 API，配合 Batch（5 折）和缓存（~1 折读取）优化。两者不互斥，很多工程师是 Max 订阅 + 生产 API 并行。

      Sonnet 5 限时价结束后会怎样？

      2026-08-31 后输入/输出从 $2/$10 恢复到 $3/$15。如果你有大批量离线任务，可以考虑在截止前用 Batch API（叠加后相当于 $1/$5）跑完。

      Pro / Max 订阅能用最强的模型吗？

      订阅档位决定的是用量额度和功能入口，可用模型以产品内实际列表为准。最新旗舰模型（如 Fable 5）的开放范围与档位要求以官方公告为准，本站不做保证性描述。
